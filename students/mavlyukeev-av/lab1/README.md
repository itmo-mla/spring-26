## Отчёт

### 1. Датасет

В качестве датасета выбран [Titanic Survival Prediction Dataset](https://www.kaggle.com/datasets/yasserh/titanic-dataset)

Данный набор данных содержит информацию о пассажирах Титаника, включая их возраст, пол, класс билета, количество родственников на борту и другие признаки.
Целевая переменная — **признак выживаемости** (1 — пассажир выжил, 0 — не выжил).

#### Подготовка данных:

- Загружен CSV-файл `titanic.csv` по URL с Kaggle;
- Использованы как числовые, так и категориальные признаки;
- Пропуски в признаках `Age` и `Embarked` сохранены и обрабатываются алгоритмом во время обучения и предсказания;
- Датасет разделён на обучающую, валидационную и тестовую выборки.

Используемые признаки:
- `Pclass` — числовой;
- `Sex` — категориальный;
- `Age` — числовой, содержит пропуски;
- `SibSp` — числовой;
- `Fare` — числовой;
- `Embarked` — категориальный, содержит пропуски.

Размер датасета: `891` объектов.

Распределение классов:
- `Did not survive`: `549` объектов (`61.6%`);
- `Survived`: `342` объекта (`38.4%`).
