## Реализованные алгоритмы

### SLIM (Sparse Linear Method)

**Идея.** Item-based подход: рейтинг пользователя `u` для айтема `i`
предсказывается как взвешенная сумма его оценок других айтемов:

```
r̂_{u,i} = Σ_{j≠i} r_{u,j} · w_{i,j}
```

Веса `w_i` обучаются отдельно для каждого айтема `i` через ElasticNet
с ограничениями:

```
min_w  (1/2) · ||r_i - R·w_i||² + (β/2)·||w_i||² + λ·||w_i||₁
       при условии: w_i ≥ 0, w_{i,i} = 0
```

- **L1-регуляризация** обеспечивает разреженность матрицы весов W.
- **L2-регуляризация** предотвращает переобучение.
- **Ограничение w_{i,i} = 0** запрещает айтему «рекомендовать самого себя».
- **Неотрицательность w_i ≥ 0** стабилизирует решение.

Задача сводится к стандартному `sklearn.linear_model.ElasticNet`, что позволяет
использовать готовые оптимизированные решатели. Задача распараллеливается по
столбцам матрицы.


### LFM — Latent Factor Model (Funk MF / SGD)

**Идея.** Матрица рейтингов R аппроксимируется как произведение двух
матриц малого ранга: R ≈ P·Qᵀ, где:
- P ∈ ℝ^{n_users × K} — матрица латентных факторов пользователей,
- Q ∈ ℝ^{n_items × K} — матрица латентных факторов айтемов.

Предсказание с bias-членами:

```
r̂_{u,i} = μ + b_u + b_i + p_u^T · q_i
```

Обучение через SGD (стохастический градиентный спуск). Ошибка:
```
e_{u,i} = r_{u,i} - r̂_{u,i}
```
Обновления на каждом шаге:
```
b_u ← b_u + γ · (e_{u,i} - λ_b · b_u)
b_i ← b_i + γ · (e_{u,i} - λ_b · b_i)
p_u ← p_u + γ · (e_{u,i} · q_i - λ_p · p_u)
q_i ← q_i + γ · (e_{u,i} · p_u - λ_q · q_i)
```

**Эталон:** `surprise.SVD` — это в точности тот же FunkMF/LFM с SGD
(документация Surprise явно указывает на алгоритм Simon Funk).

## Эталонные реализации

| Эталон                    | Библиотека      | 
|---------------------------|-----------------|
| `surprise.SVD` (Funk MF)  | scikit-surprise | 
| `KarypisSLIM`             | https://github.com/KarypisLab/SLIM |
**Установка KarypisLab/SLIM** (требует сборки из C-исходников).
 
Протестировано на Ubuntu 24.04 / WSL2 с GCC 13.
 
```bash
# 1. Системные зависимости
sudo apt-get install build-essential cmake gcc g++
 
# 2. Клонировать репозиторий (--recursive обязателен — тянет субмодуль GKlib)
git clone --recursive https://github.com/KarypisLab/SLIM.git
cd SLIM
 
# 3. Собрать GKlib (внутренняя C-библиотека)
#    Флаг -Wno-maybe-uninitialized нужен из-за GCC 13: без него сборка падает
cd lib/GKlib
make config openmp=set CFLAGS="-Wno-maybe-uninitialized"
make
cd ../../
 
# 4. Собрать и установить саму библиотеку SLIM
make config shared=1 cc=gcc cxx=gcc prefix=~/.local
make install
 
# 5. Установить Python-пакет
cd python-package
python3 setup.py install
 
# 6. Добавить путь к .so в LD_LIBRARY_PATH (нужно при каждом открытии терминала,
#    или один раз добавить в ~/.bashrc)
export LD_LIBRARY_PATH=~/.local/lib:$LD_LIBRARY_PATH
# Чтобы не вводить каждый раз:
echo 'export LD_LIBRARY_PATH=~/.local/lib:$LD_LIBRARY_PATH' >> ~/.bashrc
 
# 7. Проверка
python3 -c "import SLIM; print('SLIM установлен успешно')"
```

## Метрики качества

### RMSE
```
RMSE = sqrt( (1/|D_test|) · Σ_{(u,i)∈D_test} (r_{u,i} - r̂_{u,i})² )
```
Измеряет точность предсказания пропущенных рейтингов.

### NDCG@k (задача со *)
```
DCG_k   = Σ_{i=1}^{k} (2^rel_i - 1) / log₂(i + 1)
NDCG_k  = DCG_k / IDCG_k
```
Измеряет качество ранжирования top-k рекомендаций.
`IDCG_k` — DCG при идеальном ранжировании (верхняя граница).
