# Лабораторная работа №3: Градиентный бустинг

## Описание алгоритма

Градиентный бустинг — это ансамблевый метод машинного обучения, 
который строит модель как сумму слабых алгоритмов.

Идея алгоритма:
1. Инициализируется простая модель.
2. На каждой итерации вычисляется антиградиент функции потерь.
3. Обучается базовый алгоритм на этом антиградиенте.
4. Модель обновляется с учетом шага обучения.

Таким образом, каждая следующая модель исправляет ошибки предыдущих.

В данной работе использовалась квадратичная функция потерь (MSE).

---

## Описание датасета

Использован датасет California Housing из библиотеки sklearn.

Датасет содержит:
- признаки: характеристики районов (доход, возраст домов, население и т.д.)
- целевая переменная: медианная стоимость жилья

Размер выборки: ~20 000 объектов.
