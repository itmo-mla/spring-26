# Лабораторная 1

## Датасет
Adult Income dataset (OpenML)

## Особенности
- категориальные признаки
- числовые признаки
- добавлены искусственные пропуски (5%)

## Реализация
Реализован алгоритм ID3 с критерием Джини.

Добавлена:
- обработка пропусков
- бинаризация признаков
- редукция дерева (Reduced Error Pruning)
