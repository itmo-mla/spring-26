# Лабораторная работа №3. Градиентный бустинг

## Описание алгоритма градиентного бустинга

Градиентный бустинг — это ансамблевый алгоритм, который строит последовательность слабых моделей. Каждая следующая модель обучается исправлять ошибки предыдущих моделей.

В работе реализован бинарный градиентный бустинг для классификации:

1. Начальное предсказание задается через логит доли положительного класса.
2. На каждой итерации текущие предсказания переводятся в вероятности с помощью сигмоиды.
3. Для каждого объекта вычисляется псевдоостаток: `y - p`, где `y` — истинная метка, `p` — текущая вероятность положительного класса.
4. На псевдоостатках обучается регрессионное дерево.
5. Предсказание ансамбля обновляется с учетом скорости обучения `learning_rate`.

В качестве базовых моделей используются небольшие регрессионные деревья. Они нужны потому, что псевдоостатки являются вещественными числами, а не метками классов.

## Описание датасета

Для экспериментов используется датасет Titanic. Задача — предсказать, выжил ли пассажир.

Целевая переменная:

- `survived`: `1` — пассажир выжил, `0` — пассажир не выжил.

Использованные признаки:

- `pclass` — класс билета;
- `sex` — пол пассажира;
- `age` — возраст;
- `sibsp` — количество братьев, сестер или супругов на борту;
- `parch` — количество родителей или детей на борту;
- `fare` — стоимость билета;
- `embarked` — порт посадки;
- `alone` — путешествовал ли пассажир один.

## Результаты экспериментов

Оценка качества выполнялась с помощью стратифицированной 5-fold кросс-валидации. Метрика качества — `accuracy`.

Параметры моделей:

- `n_estimators = 150`;
- `learning_rate = 0.1`;
- `max_depth = 3`;
- `min_samples_leaf = 5`;
- `random_state = 42`.

Результаты кросс-валидации:


| Модель                                        | Accuracy, mean | Accuracy, std | Время CV, сек |
| --------------------------------------------- | -------------- | ------------- | ------------- |
| Самописный Gradient Boosting                  | 0.8294         | 0.0178        | 66.643        |
| `sklearn.ensemble.GradientBoostingClassifier` | 0.8384         | 0.0215        | 1.896         |


Дополнительная проверка на отложенной выборке `train/test = 80/20`:


| Модель                                        | Test accuracy | Время одного обучения, сек |
| --------------------------------------------- | ------------- | -------------------------- |
| Самописный Gradient Boosting                  | 0.8045        | 12.924                     |
| `sklearn.ensemble.GradientBoostingClassifier` | 0.8156        | 0.383                      |


## Сравнение с эталонной реализацией

Самописная реализация показала качество, близкое к реализации из `scikit-learn`: разница на кросс-валидации составила около `0.009` по accuracy. По времени обучения эталонная реализация значительно быстрее. На тестовой выборке результаты тоже близки: разница на отложенной выборке составила около `0.011` по accuracy (`0.8045` у самописной модели против `0.8156` у `scikit-learn`). Время одного обучения на train-части у эталона также заметно меньше: `0.383` сек против `12.924` сек.
