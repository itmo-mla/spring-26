# Лабораторная работа №4. EM-алгоритм

## Описание наивного байесовского классификатора

Наивный байесовский классификатор основан на формуле Байеса и предположении условной независимости признаков при фиксированном классе:

```text
P(y | x) ~ P(y) * product P(x_j | y)
```

Для каждого класса оцениваются априорная вероятность класса и распределения признаков. Объект относится к классу с максимальной апостериорной вероятностью. В данной лабораторной работе основной алгоритм другой: GMM восстанавливает плотность распределения данных без использования меток.

## Описание алгоритма

Gaussian Mixture Model представляет плотность как смесь нормальных распределений:

```text
p(x) = sum pi_k * N(x | mu_k, Sigma_k)
```

Параметры модели:

- `pi_k` — вес компоненты;
- `mu_k` — среднее компоненты;
- `Sigma_k` — ковариационная матрица.

EM-алгоритм состоит из двух шагов:

- E-step: вычисляются ответственности компонент `gamma_ik`;
- M-step: обновляются `pi_k`, `mu_k`, `Sigma_k`.

Качество оценивается через среднее лог-правдоподобие, то есть по принципу максимального правдоподобия.

## Описание датасета

Использован датасет Iris из `sklearn.datasets`.

- Количество объектов: `150`;
- количество признаков: `4`;
- количество классов: `3`;
- классы: `setosa`, `versicolor`, `virginica`.

Перед обучением признаки были стандартизированы с помощью `StandardScaler`. Метки классов использовались только для оценки качества кластеризации.

## Результаты экспериментов

Основной эксперимент проводился при `K = 3`, так как в датасете три класса.


| Модель                    | Mean log-likelihood | Accuracy   | ARI        | NMI        | Итерации |
| ------------------------- | ------------------- | ---------- | ---------- | ---------- | -------- |
| Самописная GMM            | `-1.936875`         | `0.966667` | `0.903874` | `0.899694` | `18`     |
| `sklearn GaussianMixture` | `-1.936874`         | `0.966667` | `0.903874` | `0.899694` | `23`     |


Дополнительно проверялось разное число компонент `K = 2..6`. Лучшее соответствие реальным классам по ARI и NMI получилось при `K = 3`.

## Сравнение с эталонной реализацией

Самописная реализация дала практически тот же результат, что и sklearn: лог-правдоподобие отличается только в шестом знаке после запятой, а `accuracy`, `ARI` и `NMI` совпали.

Это показывает, что реализованные E-step и M-step работают корректно. Для устойчивости использовались вычисления в логарифмах, регуляризация ковариационных матриц и k-means инициализация.

## Выводы

В ходе работы была реализована модель GMM, обучаемая EM-алгоритмом. На датасете Iris модель смогла хорошо восстановить структуру данных: точность после сопоставления компонент классам составила около `96.7%`.

Сравнение с scikit-learn показало, что самописная реализация воспроизводит поведение эталонной модели и достигает такого же качества на выбранном датасете.
