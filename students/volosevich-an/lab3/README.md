## Методология

### 1. Реализация базового алгоритма

В качестве базового алгоритма используется кастомное CART-дерево решений (регрессионное дерево), адаптированное под задачу многоклассовой классификации (price_range ∈ {0,1,2,3}).

Дерево обучается на псевдо-остатках градиентного бустинга и используется как слабый базовый алгоритм b_t(x) в соответствии с лекционной схемой градиентного бустинга.

Эталонным аналогом является реализация `DecisionTreeClassifier` / `GradientBoostingClassifier` из scikit-learn.


---

### 2. Градиентный бустинг

Используется классический градиентный бустинг для многоклассовой классификации:

- Аддитивная модель:
  - a_t(x) = a_{t-1}(x) + η · α_t · b_t(x)

- Функция потерь:
  - multinomial logistic loss (softmax)

- Антиградиент:
  - g_i = y_i - p_i

Каждая новая модель обучается на исправлении ошибок предыдущих моделей.


---

### 3. Стохастический бустинг

Для улучшения обобщающей способности используется subsample:

- случайная подвыборка объектов (subsample = 0.8)
- снижение корреляции между базовыми моделями
- уменьшение переобучения


---

### 4. Оптимизация шага (learning rate scaling)

Дополнительно используется коэффициент α_t, который подбирается через одномерную минимизацию функции потерь:

- α_t = argmin L(a_{t-1} + α · b_t)

что соответствует лекционной постановке градиентного бустинга.


---

### 5. Агрегация моделей

Финальное предсказание строится как сумма вкладов всех базовых моделей:

- F(x) = Σ η · α_t · b_t(x)

и последующее применение softmax для получения вероятностей классов.


---

## Результаты

### Параметры модели

```python
model_params = {
    "n_estimators": 25,
    "learning_rate": 0.1,
    "max_depth": 3,
    "subsample": 0.8,
    "random_state": 42
}
n_splits = 5
```


### Результаты экспериментов
#### Custom Gradient Boosting
- mean_accuracy = 0.8395
- std_accuracy = 0.0283
- mean_train_time = 6.9638 s


#### Sklearn Gradient Boosting
- mean_accuracy = 0.8430
- std_accuracy = 0.0251
- mean_train_time = 0.2482 s

### Анализ результатов
- Качество кастомной реализации близко к sklearn (разница ~0.0035), что подтверждает корректность реализации алгоритма;
- Основное отличие наблюдается во времени обучения из-за отсутствия оптимизаций уровня библиотеки scikit-learn;
- Стохастическая подвыборка (subsample) положительно влияет на устойчивость модели;
- Увеличение числа деревьев повышает качество, но также увеличивает время обучения;
- Реализация корректно воспроизводит поведение классического градиентного бустинга для многоклассовой задачи.
