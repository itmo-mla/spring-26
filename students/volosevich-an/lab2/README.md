## Методология



### 1. Реализация базового алгоритма

Базовый алгоритм ID3 дерево из лабораторной работы № 1, результаты работы которого качественно сравнимы с эталонным алгоритмом DecisionTreeClassifier.

Забегая наперед собственный алгоритм RandomForest покажет сравнимое качество с эталонным алгоритмом из scikit-learn.


### 2. Беггинг

Каждое дерево обучается на **bootstrap выборке**.

### 3. Подмножество признаков

Подмножество признаков выбирается случайным образом, доступы следующие конфигурации:
- `sqrt(n_features)`
- `log2(n_features)`

### 4. Агрегация и оптимизация
Голосование по формуле:

- F(x) = (1 / T) * Σ bt(x)

Для каждого объекта используется только часть деревьев:

- OOB(x_i) = (1 / |Ti|) * Σ bt(x_i)

где:
- Ti — деревья, не обучавшиеся на x_i

Важность признаков вычисляется как:

- importance_j = (OOB^j - OOB) / OOB

где:
- OOB^j — ошибка после перемешивания признака j

Оптимизация по **OOB accuracy**.
