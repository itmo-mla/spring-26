# Лабораторная работа № 5

## Цель работы
Реализовать алгоритм Sparse Linear Method (SLIM) для рекомендательных систем, сравнить его с эталонной реализацией на базе `ElasticNet` из `scikit-learn`, реализовать латентную семантическую модель и сравнить её с эталонной реализацией `TruncatedSVD` из `scikit-learn`.

## Выбранный датасет
Для работы использован текстовый рекомендательный датасет с материалами по темам: computer_vision, databases, machine_learning, natural_language, recommenders, security.

Датасет содержит:
- пользователей: 24;
- текстовых объектов: 36;
- пользовательских оценок: 408;
- обучающих взаимодействий: 336;
- тестовых взаимодействий: 72.

Каждый объект имеет тему, заголовок и текстовое описание. Пользователи оценивают материалы по шкале от 1 до 5. Задача модели — восстановить скрытые оценки и сформировать ранжированный список рекомендаций.

## Подготовка данных
Данные разделены по пользователям: для каждого пользователя часть оценок оставлена в тестовой выборке, остальные оценки используются для обучения. На основе обучающих взаимодействий построена user-item матрица, где строки соответствуют пользователям, столбцы — текстовым объектам, а значения — оценкам.

Качество оценивается двумя метриками:
- `RMSE` — ошибка восстановления тестовых оценок;
- `NDCG@10` — качество ранжирования рекомендаций в первой десятке.

## Описание SLIM
SLIM строит разреженную item-item модель. Для каждого объекта обучается линейная регрессия, которая восстанавливает оценки данного объекта через оценки остальных объектов:

\[
\hat R = RW
\]

где `R` — user-item матрица, а `W` — матрица item-item коэффициентов.

Оптимизационная задача для одного объекта:

\[
\frac{1}{2n}\|r_j - Rw_j\|_2^2 + \lambda_1\|w_j\|_1 + \frac{\lambda_2}{2}\|w_j\|_2^2
\]

Дополнительно диагональный коэффициент `w_jj` зануляется, чтобы объект не предсказывал сам себя. В собственной реализации задача решается методом ISTA: выполняется градиентный шаг по квадратичной части, затем применяется soft-thresholding для L1-регуляризации и ограничение неотрицательности коэффициентов.

Эталонная SLIM-модель реализована через `ElasticNet` из `scikit-learn`: для каждого объекта обучается отдельная регуляризованная линейная модель с теми же гиперпараметрами.

## Описание латентной семантической модели
В качестве латентной семантической модели реализована модель на основе усечённого SVD. User-item матрица приближённо раскладывается на низкоранговые факторы:

\[
R \approx U_k \Sigma_k V_k^T
\]

где `k` — число латентных компонент. Такая модель восстанавливает скрытые предпочтения пользователя через латентные факторы и позволяет предсказывать оценки для ещё не просмотренных объектов.

Собственная реализация использует `numpy.linalg.svd`. Эталонная реализация использует `TruncatedSVD` из `scikit-learn`.
