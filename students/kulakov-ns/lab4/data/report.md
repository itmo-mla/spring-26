# Лабораторная работа № 4

## Цель работы
Реализовать EM-алгоритм для гауссовой смеси, обучить модель на датасете Iris, оценить качество восстановления плотности и сравнить результат с эталонной реализацией `GaussianMixture` из `scikit-learn`.

## Выбранный датасет
Для работы использован датасет Iris.

Причины выбора:
- все признаки непрерывные, что удобно для гауссовой смеси;
- в датасете есть 3 естественных класса, что позволяет дополнительно оценить качество кластеризации;
- датасет небольшой, поэтому удобно анализировать работу EM-алгоритма и сравнивать собственную реализацию с библиотечной.

## Подготовка данных
Использованы признаки:
`sepal length (cm), sepal width (cm), petal length (cm), petal width (cm)`.

Предобработка сделана в стиле предыдущих лабораторных:
- данные разделены на обучающую и тестовую выборки в отношении 80/20 со стратификацией по классам;
- числовые признаки стандартизованы через `StandardScaler`.

## Описание метода
Собственная реализация построена как `GaussianMixtureCustom`, обучаемая EM-алгоритмом.

Алгоритм работает следующим образом:
1. Инициализируются веса компонент, средние и ковариационные матрицы.
2. На E-шаге вычисляются апостериорные вероятности принадлежности объектов компонентам смеси.
3. На M-шаге по полученным вероятностям переоцениваются веса, средние и ковариации.
4. Шаги повторяются до сходимости по изменению среднего логарифмического правдоподобия.
5. Лучшая модель выбирается по среднему логарифмическому правдоподобию на кросс-валидации.

Подбор гиперпараметров выполнялся перебором по сетке параметров с 5-блочной `KFold`-кросс-валидацией. В качестве основной метрики использовалось среднее логарифмическое правдоподобие (`avg log-likelihood`), что соответствует задаче восстановления плотности. Метрики `accuracy` и `ARI` приведены дополнительно, так как у датасета есть истинные метки классов.
