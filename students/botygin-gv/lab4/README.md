## Наивный байесовский классификатор
Генеративная модель: $a(x) = \arg\max_y P(y) \prod_j p(x_j|y)$.  
**Допущение**: условная независимость признаков.  
**Плюсы**: быстро, устойчиво к шуму. **Минусы**: смещение при коррелированных признаках.

## Датасет

- 800 объектов, 2 признака, 3 кластера
- Разная дисперсия компонент: $[1.0, 2.5, 5.0]$ — проверка устойчивости к разномасштабным кластерам
- Истинные метки известны только для оценки качества (ARI)
