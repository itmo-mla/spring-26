# Лабораторная работа №3. Градиентный бустинг

## Задание
- Выбрать датасет для анализа;
- Реализовать алгоритм градиентного бустинга.
- Обучить модель на выбранном датасете.
- Оценить качество модели с использованием кросс-валидации.
- Замерить время обучения модели.
- Сравнить результаты с эталонными реализациями из библиотеки scikit-learn:
  - точность модели;
  - время обучения;
- Подготовить отчет, включающий:
  - описание выбранного метода;
  - описание датасета;
  - результаты экспериментов;
  - сравнение с эталонными реализациями;
  - выводы.

## Исходные данные
**Датасет:** Breast Cancer  
**Кросс-валидация:** 5 folds, стратифицированная  
**Гиперпараметры:** `n_estimators=100`, `learning_rate=0.1`, `max_depth=3`

## Результаты

| Модель | Accuracy | Время обучения (с/фолд) |
|--------|----------|------------------------|
| Кастомная реализация | 0.9403 ± 0.0290 | 0.2406 |
| sklearn.ensemble | 0.9491 ± 0.0237 | 0.2547 |

## Выводы

1. **Точность:** реализация из sklearn показала результат на 0.0088 выше. Разница небольшая, что говорит о корректности реализации базового алгоритма.

2. **Время обучения:** на малом датасете разница несущественна. Кастомная версия даже чуть быстрее, но на больших данных преимущество оптимизированной sklearn-реализации станет заметнее.
