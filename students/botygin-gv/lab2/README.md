# Лабораторная работа №2. Линейные ансамбли

## Задание
- Выбрать датасет для анализа;
- Реализовать метод случайных подпространств (RSM) или Random Forest;
- Обучить ансамбль, подобрать оптимальные гипер-параметры. Для подбора оптимальных параметров использовать grid search из sklearn; Оптимальные параметры подбирать по OOB;
- Получить оценку важности признаков через OOB^j
- Сравнить результаты с эталонными реализациями из библиотеки scikit-learn:
  - точность модели;
  - время обучения;
- Подготовить отчет, включающий:
  - описание выбранного метода;
  - описание датасета;
  - результаты экспериментов;
  - сравнение с эталонными реализациями;
  - выводы.

## Датасет
Встроенный датасет `sklearn.datasets.load_breast_cancer()`. 569 объектов, 30 количественных признаков, задача бинарной классификации.

## Результаты
Разделение: 80% train / 20% test.

- **Оптимальные параметры:** `n_estimators=50`, `max_features='log2'`, `max_depth=10`, `bootstrap=True`
- **Custom RF:** Test Acc = 0.9474, OOB = 0.9473, время = 0.297 с
- **Sklearn RF:** Test Acc = 0.9561, OOB = 0.9604, время = 0.060 с
- **Топ признаков (OOB^j):** `worst fractal dimension` (16.67%), `worst concave points` (8.33%), `worst smoothness` (8.33%)

## Выводы
Собственная реализация показала сопоставимую с эталонной точность и корректно вычислила важность признаков по формуле OOB^j. Разница во времени обучения (~5×) объясняется отсутствием низкоуровневых оптимизаций в пользовательском коде.
