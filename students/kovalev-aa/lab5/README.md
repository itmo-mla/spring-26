# Отчёт: SLIM и латентная семантическая модель

## 1. Постановка задачи

- Реализовать SLIM и латентную семантическую модель (матричную факторизацию)
- Обучить на датасете MovieLens
- Оценить качество по **RMSE**, **MAE**, **NDCG@10**
- Сравнить с эталонными реализациями

---

## 2. Описание моделей

### 2.1 SLIM (Sparse Linear Methods)

Предсказание для пользователя u и item i строится как линейная комбинация оценок пользователя по похожим item-ам:

**r̂_ui = Σⱼ≠ᵢ wᵢⱼ · r_uj**

Матрица весов W обучается минимизацией:

**||X - X·W||² + α·||W||₁ + β·||W||²**

где α и β — коэффициенты L1 и L2 регуляризации.

### 2.2 Латентная семантическая модель (SVD)

Матричная факторизация с bias-компонентами:

**r̂_ui = μ + b_u + b_i + p_uᵀ·q_i**

где:
- μ — глобальное среднее
- b_u, b_i — смещения пользователя и item-а
- p_u, q_i — латентные факторы

Обучение — стохастический градиентный спуск.

## 6. Выводы

1. **SLIM** реализован корректно.

2. **Латентная семантическая модель** значительно лучше SLIM по всем метрикам.

4. **NDCG@10** показывает, что SLIM плохо подходит для ранжирования, в отличие от факторизационных моделей.
